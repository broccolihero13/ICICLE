ICICLE - Integrated Canvas Improved Click Less Environment
===========================

This extension is in the very early beta phase. If you are utilizing this tool,
you may have trouble finding support when it doesn't work. Canvas support will
not have any information regarding this tool.

Current Functionality:
-Shows SIS ID on object pages (i.e. User SIS IDs on the "/accounts/self/users" path)
-Inactivate or Conclude enrollments for an individual user (checkbox exclude to target specific enrollments)


This tool will only be available when in your *.instructure.com account
